Password akun Microsoft 365 Anda sudah direset.

Format Password baru: #AlwaysTheFirst#password_sion

Contoh:
Password sion : abcd1234
Password baru : #AlwaysTheFirst#abcd1234

Demi keamanan akun Anda, silakan ganti password anda setelah login. Info lebih lanjut cek FAQ Reset Password di SION.
